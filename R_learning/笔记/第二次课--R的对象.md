rep() 重复函数
```r
data4 <- rep(2:5, times = 4) # 重复函数
```

```r

```

```r
data5 <- seq(from = 3, to = 21, by = 3) # 序列函数
```
输出：  
```r
3 6 9 12 15 18 21
```

### 矩阵运算
- 矩阵的加法（A+B）
- 矩阵的减法（A-B）
- 矩阵的各元素的乘法（A*B）
- 矩阵相乘（A%*%C）
- 矩阵转置 t()  
例如：t(A)  
- 矩阵求解 solve()  
例如：求解DX=A中的X，可以使用solve(D,A)
- 矩阵的特征值和特征向量 eigen()  
例如：E<-eigen(D)
